# 🚀 3D Custom Print Shop & Shopier Integration

  - Bu proje; kişiselleştirilmiş 3D baskı ürünlerinin sergilendiği, HTML/CSS/JS arayüzü geliştirilmiş. ASP.NET Core ile yapılandırılmış bir backend sisteminden oluşan uçtan uca bir e-ticaret çözümüdür.

🛠 Teknoloji Stack
- Frontend: HTML, Javascript, React

- Backend: ASP.NET Core Web API (C#) (Eklenicek)

- Veritabanı: SQL Server (EF Core) (Eklenicek)

- Ödeme: Shopier (Webhook & Redirect)

- Güvenlik: JWT Authentication & CORS Management (Eklenicek)
